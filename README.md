# 문자열 계산기

## 프로그래밍 요구사항

- indent는 2를 넘지 않는다. 다음과 같이 작성하는 경우 indent는 3이다.  
```java
    for (Foo foo : List<Foo> foos) {
        if (foo == null) {
            throw new IllegalArgumentException();
        }
    }
```
- `else`, `switch ~ case` 예약어를 사용하지 않는다.
- 메서드의 길이는 15라인을 넘어가지 않는다.
- setter를 사용하지 않는다.
